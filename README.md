# Kodeview

Kodeview is a lightweight React Native / Expo client for browsing code repositories with a focus on readable rendering of code and markdown.

## Features

- Browse repositories, branches and commits
- View files with syntax-highlighted code blocks
- Read rendered markdown with images

## Getting Started

Prerequisites:

- Node.js 18+ and npm or Yarn
- Expo CLI (optional but recommended)

Install:

```bash
npm install
# or
yarn
```

Run (development):

```bash
npx expo start
# or
npm run start
```

Build:

Follow Expo build/publish instructions for your target platform.
